Instructions

Dear participant , please read the instructions before the experiment.

Experiment length: About 40 minutes

Practice Trial: A practice trial would be administered to familiarize yourself with the experiment setup and the copy-paste techniques.

The experiment: will consists of 27 trials of copy-and-pasting for each technique (Traditional, AutoComPaste). You will be given a one minute break every time you complete one experiment set which consists of 54 trials for both techniques. We will repeat each experiment set 3 times.

Please watch the AutoComPaste demo video to learn more about it.
